Model: each phase (martensite, bainite, pearlite, ferrite) has a characteristic cooling-rate "sweet spot" set by the alloy's critical cooling rate (CCR). Phase fractions are Gaussian weights in log-cooling-rate space, normalized to 100%. This is a simplified teaching model, not a substitute for a measured CCT/TTT diagram.

QuenchLab — companion project to Alloy Atlas. The same phase-fraction model backing this page is implemented in quenchlab_analysis.py for batch analysis and CCT-style plotting.
